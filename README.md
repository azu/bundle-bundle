# bundle-bundle

requireしてbundleするだけ

## Installation

    yarn install

## Usage

    yarn run build

- index.js - 読み込む + global(window)に追加する
- bundle.js - `yarn run build`でビルドしたもの
- index.html - bundle.jsを読み込んでる
